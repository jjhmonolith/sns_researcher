## 핵심 내용

- 사실: 조희진은 korean-law-mcp 프로젝트를 통해 법령, 판례, 행정규칙, 자치법규, 조약, 해석례를 AI assistant나 terminal에서 바로 사용할 수 있다고 소개했다.
- 사실: 법제처 Open API 인증키를 받으면 Claude 웹, Claude desktop app, terminal 등에서 활용 가능하다고 설명했다.
- 의견: 작성자는 인사·노무 업무에서 수동 검색 시간을 줄이고, 일상어 기반으로 더 정확한 검색이 가능해질 것이라고 기대했다.

## 시사점

- AX 관점에서는 전문지식 업무 자동화의 핵심이 '모델 교체'보다도 신뢰 가능한 외부 지식원 연결과 검색 인터페이스 전환에 있음을 보여준다.
- 기업은 법무·인사·컴플라이언스 영역에서 MCP 같은 connector 계층을 통해 AI를 기존 정보원 위에 얹는 방식으로 빠른 실험이 가능하다.
- 기존 지식 연결: '[seedling] 기업 워크플로우형 Claude 역량의 핵심이 연결·평가·배포 운영이라는 관점'과 직접 연결된다. 특히 연결(connectivity)이 실무 가치의 핵심이라는 점이 같다.
