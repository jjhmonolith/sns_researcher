## 원문

AI가 콘텐츠는 만들어줘도,
우리 브랜드답게 디자인을 만들어주진 않습니다.
저는 이 간극을 Figma 디자인 시스템 + Claude code
두 가지 조합으로 해결해봤어요.
구현 방식은 이렇습니다
1.Figma 토큰·컴포넌트로 디자인 시스템 구축 →
Claude Code로 코드화
2.Figma 플러그인으로 기존 템플릿 학습
3.Claude API 연동 → 기획 입력→
Figma 템플릿에 반영
4.Figma 안에서 컬러 모드
컴포넌트 수정으로 유지보수
"그냥 Claude Code + Figma MCP가 더 싸지 않나요?"
맞습니다. 비용만 보면
MCP 직접 연결이 효율적이에요.
하지만 MCP 같은 도구에
익숙하지 않은 팀원과 함께 쓴다면,
누구나 쓸 수 있는 플러그인 형태가
훨씬 현실적일것이라 생각했어요.
조직에서 일관된 브랜드 디자인으로
템플릿을 만들어야 하는 분들이 많을 것 같아,
만드는 과정을 영상으로 자세히 남겨두었습니다.
