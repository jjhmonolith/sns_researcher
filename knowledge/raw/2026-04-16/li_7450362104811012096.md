## 원문

매일 아침 4개 도구를 열어보던 루틴을 Slack 한 줄로 대체했습니다.
GA 열고, Sentry 열고, Clarity 열고, DB에 접속하고.
각 3분씩 매일 12분, 한 달이면 4시간.
이 반복 작업을 Claude Agent SDK로 자동화한 과정을 공유합니다.
256줄의 코드로 완성한 에이전트에서 배운 3가지입니다.
1. MCP 4개를 연결하면 Claude가 어떤 도구에서 뭘 가져올지 스스로 판단합니다.
GA4, Sentry, Clarity, PostgreSQL을 MCP로 연결했습니다. "@bot 하루 요약"이라고 말하면 Claude가 4개 데이터 소스를 순회하며 필요한 정보를 자율적으로 수집합니다.
2. 전체 코드가 src/index.ts 단일 파일 256줄입니다.
Claude Agent SDK의 설계 철학 덕분에 복잡한 오케스트레이션 코드가 필요 없습니다. MCP 서버 설정과 Slack 연동 코드가 전부입니다.
3. 도구를 줄이는 것이 아니라, 도구를 연결하는 것이 자동화입니다.
GA, Sentry, Clarity, DB를 각각 쓰는 건 같습니다. 이 4개를 하나의 인터페이스로 연결하는 것이 핵심입니다. Slack이 그 인터페이스가 되었습니다.
