## 원문

Anthropic leaked 500,000 lines of code.
It went crazy viral.
In a strange way, it also created an accidental window into how Anthropic builds.
Let's start with, what kind of code was actually leaked?
Reportedly, what was exposed was a large part of the internal TypeScript source code behind Claude Code - not model weights.
In other words, this was a partial look into how a top AI company builds a production-grade AI product around the model.
People are now reverse-engineering it, analyzing the structure, and using it to better understand how a production-grade coding agent is put together.
And that is valuable for a reason people outside product teams often underestimate:
- code like this is never just code.
Behind it are countless discussions across engineering, product, design, safety, infrastructure, and leadership.
A lot of tradeoffs.
A lot of decisions.
A lot of “why did they build it this way instead of that way?”
So when source code like this becomes visible, what people are really studying is not just implementation.
They are studying product thinking made concrete.
Also, my favorite little detail:
in the middle of all this serious AI coding infrastructure, people found a Tamagotchi-style virtual pet in the leaked code.
Completely non-essential to the bigger story.
But somehow very cute.
I like this detail so much - maybe because it reminds me that even inside very serious AI products, there are still small human touches.
