1️⃣ 첫 번째 시도 — Claude Code MCP를 Figma에 연결하기
1. Claude Code MCP를 Figma에 연결
2. AI로 Figma에 직접 디자인해달라고 요청
3. 결과 — 실패 😭
Figma MCP는 읽기 전용이었어요. 디자인을 읽어서 코드로 변환하는 용도이고, 캔버스에 직접 그리는 건 현재 MCP 연결 방식으로는 안 돼요.
(혹시 방법 아시는 분 알려주세요!)
2️⃣ 두 번째 시도 — Claude Code로 Figma 플러그인 만들기
1. Claude Code에게 “Figma에 디자인 그려주는 플러그인 만들어줘” 요청
2. Claude Code가 플러그인 파일 자동 생성 → Figma에 설치
3. 결과 — 실제로 그려졌어요! 근데… 한계가 있었어요.
플러그인 설치 과정은 생각보다 쉬웠어요. 그러나 제 입장에서 결과가 약간 엉성(?)했어요. 완성도 측면에서 좀 아쉽더라구요.
왜 이런지 디테일하게 파볼까 했는데 사실 플러그인 까는 것보다 더 편한 방법이 있을 것 같다는 생각이 들었어요.
3️⃣ 세 번째 시도 — Paper 사용하기 ✅
Figma 대신 Paper 를 사용해봤어요
Paper의 핵심은 디자인 = 코드라는 점이에요.
Figma는 디자인 후 개발자가 코드로 다시 만들어야 하지만, Paper는 캔버스 자체가 HTML/CSS라 디자인한 게 그대로 코드가 돼요.
1. Claude Code MCP를 Paper에 연결
2. 터미널에 입력: “다크 배경에 히어로 섹션 만들어줘. 굵은 헤드라인, 서브타이틀, 그라디언트 CTA 버튼 포함.”
3. 결과 — 캔버스를 한 번도 건드리지 않았는데 화면이 그려졌어요.(영상 첨부)

I finally got AI to design for me!!!!
I'm a UX/UI Designer.
My goal with AI has always been clear:
Get AI to design in Figma instead of me!
해시태그
#Paper might just make that possible.
Here's my honest journey of trying to make AI-powered auto-design work. 🙂
1️⃣ First Attempt — Connecting Claude Code MCP to Figma
1. Connected Claude Code MCP to Figma
2. Asked AI to design directly on the Figma canvas
3. Result — Failed 😭
Figma MCP is read-only. It's built for reading designs and converting them to code — not for drawing directly on the canvas with the current MCP setup.
(If anyone knows a way around this, please let me know!)
2️⃣ Second Attempt — Building a Figma Plugin with Claude Code
1. Asked Claude Code: "Make me a plugin that draws designs in Figma"
2. Claude Code auto-generated the plugin file → installed it in Figma
3. Result — It actually drew something! But… there were limits.
The installation was easier than expected. But the output felt a bit rough around the edges — not quite the quality I was hoping for. I started wondering if there was a smoother way to do this without dealing with plugin setup at all.
3️⃣ Third Attempt — Using “Paper” ✅
I tried Paper instead of Figma.
The core idea of Paper: design = code.
1. Connected Claude Code MCP to Paper
2. Typed into the terminal: "Create a hero section with a dark background. Bold headline, subtitle, and a gradient CTA button."
3. Result — A screen appeared without me touching the canvas once. (video attached)
