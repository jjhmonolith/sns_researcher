## 원문

Claude Design이 피그마의 시가총액 8000억을 증발시켰습니다. 그리고 써봤는데.. 이거 그냥 피그마 아니고, Generative UI 인데요? 모호한 요청을 던지니까 저한테 질문하는 form 을 UI로 생성해버립니다. 심지어 다양한 와이어프레임을 만들어달라고 하니까 탭 6개로 프로토타입을 나눠줬어요. 신기하지만 조심하세요. 토큰이 녹아요 ㅎㅎㅎ
1. 오직 html과 css
재밌는 건 figma 도 pencil 도 그 제품만의 확장자 파일을 만들어서 관리해버리는데 여긴 그냥 HTML과 CSS를 사용합니다. 새로운 확장자 같은 건 필요 없어요. AI가 가장 잘 하는 형식으로 뽑아버리면 됩니다.
2. prototype, slide deck, from template - Canva도 죽였습니다.
저는 이 중에 prototype을 집중해서 써봤는데요. GenUI로 form 도 보여주고, prototype을 6개 만드는데, 이걸 한 화면에 6개 탭으로 구분해서 만들어줘버리는게 충격적이었습니다. 일반적인 상식과 다르잖아요?
3. Claude Code / Codex 를 잘 쓰고 있던 사람이라면 굳이..?
내 하네스를 잘 만들어서 사용하고 있었다면 굳이.. 쓸까 싶습니다. 그런데 슬라이드 덱이나 프로토타입을 Figma, Canva를 사용하고 있던 것들은 바꿀 것 같아요. 저도 ironclaw meetup 을 준비하면서 html로 만들어서 배포한 발표자료와 figma slides 를 2개 썼거든요. figma slides 로 직접 사진을 붙여넣는게 더 빨라서 그렇게 했는데 앞으로는 이 기능을 써볼 것 같아요.
결국 Anthropic에서 만든 하네스네요. 벌써 몇 시간 지났으니 Claude Design의 opensource alternatives 가 나올거고 그것도 테스트해보면 재밌을 것 같습니다. 그리고 아마 figma, canva, pencil 에서도 같은 기능을 내겠죠? 코드의 가치가 0이 돼서 벌어지는 현상이 재미있고 또 무섭네요
