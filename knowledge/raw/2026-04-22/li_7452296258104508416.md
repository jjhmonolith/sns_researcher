## 원문

블룸버그 터미널 연 구독료 약 3만 2천 달러. 한화 4,600만원 수준입니다.
같은 역할을 지향하는 오픈소스 대안이 한 명의 개발자 손에서 나와 깃허브 별 4만개를 넘겼습니다. 이번 주말 개인 서버에 설치해볼 예정이라, 공부한 내용을 정리해 공유합니다.
▶ 프로젝트: World Monitor
실시간 글로벌 인텔리전스 대시보드입니다. 한 화면에 다음 데이터가 통합됩니다.
・지진·기상·자연재해 실시간
・산불 확산 경로 (NASA FIRMS 위성)
・선박 궤적 (AIS 스트림)
・항공기 추적 (ADS-B)
・92개 거래소 주식, 원자재, 크립토
・뉴스 피드 435개를 AI가 묶어 브리핑
・12개 신호를 합친 국가 리스크 점수
・군사·경제·재난 신호 교차 분석
3D 지구본과 WebGL 평면 지도, 두 엔진을 함께 제공합니다. 지원 언어 21개, 변형 사이트 5종.
▶ 무료 구조의 설계
프로덕트가 “무료 가입 친화적 오픈소스” 모델로 정교하게 짜여 있습니다.
1. API 키 없이 즉시 사용
지진, 기상, 난민 이동, 예측시장, 크립토, 해저 케이블, 사이버 위협 데이터는 공개 엔드포인트 기반으로 설치 직후 동작합니다.
2. 무료 가입으로 확장되는 영역
GROQ(일 14,400건 AI 추론), FRED(연준 경제 데이터), NASA FIRMS, AISStream, Finnhub, ACLED 등 9개 서비스가 무료 티어로 연결됩니다.
3. 유료 의존성은 단 하나
Cloudflare Radar(인터넷 장애 감지)뿐이며, 사용하지 않아도 나머지 기능 전체가 정상 동작합니다.
AI 브리핑은 Ollama, vLLM 등 OpenAI 호환 로컬 LLM으로 대체 가능합니다. 외부 API 의존을 완전히 제거할 수 있는 구조입니다.
▶ 구동 방식
git clone https://lnkd.in/gH9XD3KF
cd worldmonitor
docker compose up -d
./scripts/run-seeders.sh
Docker Compose 스택이 4개 컨테이너(nginx+Node.js API, Redis, Redis REST 프록시, AIS 릴레이)로 로컬 3000 포트에 떠오릅니다. API 키 9개 발급 시간을 포함해 2~3시간이면 구축이 끝날 것으로 보입니다.
▶ 기술적으로 인상적인 부분
이 프로젝트는 주 개발자 한 명이 주도해 왔습니다. 커밋 2,700건 이상, 릴리스 43번, 지금도 매주 업데이트가 올라옵니다. 리포 루트에 AGENTS.md 파일이 존재하는 것으로 보아 에이전트 기반 개발 도구를 일정 비중 활용한 것으로 추정됩니다.
▶ AX 관점의 시사점 세 가지
첫째, 정보 인프라 비용 구조가 재설계되고 있습니다.
연 수천만원 규모의 구독 비용 항목이 “로컬 서버 전기세”로 이동할 수 있는 케이스가 실제로 존재합니다.
둘째, 의사결정용 UI가 민주화되고 있습니다.
지금까지 조직의 상황 인식 도구는 구독 예산에 종속되어 있었습니다. 이제 “우리 조직 전용 상황판을 직접 설계할 것인가”가 실무 질문이 됩니다.
셋째, 1인 레버리지의 상한이 다시 올라갔습니다.
블룸버그 스타일 UI를 한 명이 구축 가능한 현실. AX 의사결정에서 “조직 규모”의 의미가 바뀌는 지점입니다.
▶ 먼저 확인한 제약
・한국어 뉴스 피드 커버리지가 약합니다. 영어권 중심 설계.
・AGPL-3.0 라이선스. 개인·자가 호스팅은 자유, 상용 SaaS나 리브랜딩은 별도 라이선스 계약이 필요합니다.
・Redis 볼륨 삭제 시 데이터 재시드가 필요합니다.
・항공 ADS-B, 선박 AIS 등 일부 데이터는 외부 API 가용성에 의존합니다.
귀사의 “월요일 아침 상황판”에는 지금 무엇이 올라와 있습니까. 새로 설계한다면, 어떤 신호가 제일 먼저 필요할까요.
이 질문에 대한 답이, 2026년 조직 의사결정 인프라의 격차를 만들 것이라 봅니다.
