## AX 관련성
기업 AX에서 agent memory, 지식 연결, 지속 컨텍스트 공유를 제품 레이어로 구현하는 방향을 보여준다.

## 주요 관점
- 에이전트 시대의 병목은 모델이나 도구보다 지속적이고 공유 가능한 memory라고 본다.
- 메모리는 silo에 갇히면 안 되며 여러 agent와 채널이 공통 컨텍스트로 활용할 수 있어야 한다고 본다.
- 기존 memory stack(Mem0, Zep, Neo4j 등)은 일반 사용자에게 너무 복잡하므로 no-setup형 memory layer가 필요하다고 본다.

## 주목할 포스트
