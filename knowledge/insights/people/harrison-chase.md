# Harrison Chase

**소속/직함**: CEO, LangChain


## AX 분야 관련성
에이전트 기반 AX 논의에서 메모리 소유권, 컨텍스트 자산, 플랫폼 종속성 문제를 구조적으로 설명하는 핵심 인물로 언급된다.


## 주요 관점

- 에이전트 시스템의 학습은 Model, Harness, Context의 3개 레이어에서 일어난다고 본다.
- 실질적인 학습과 경쟁우위는 context layer, 즉 memory에서 발생한다고 주장한다.
- genuine memory ownership requires open harness architecture라는 관점에서, 메모리 소유권을 위해 오픈 harness가 필요하다고 본다.
- 이 프레임을 바탕으로 자사 에이전트 제품 포지셔닝을 강화하고 있다.


## 주목할 포스트

- AI 에이전트 시대의 경쟁우위를 Model/Harness/Context 레이어로 설명하고, memory ownership과 open harness의 중요성을 주장한 연속 글.
