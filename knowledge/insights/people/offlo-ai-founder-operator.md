# Offlo.ai 창업자/운영자

**소속/직함**: Founder/Operator, Offlo.ai
**업데이트**: 2026-04-16


## AX 관련성
AX 관점에서 고객지원·세일즈 조직의 운영 모델을 에이전트 중심으로 재설계하는 실무적 시각을 제공한다.

## 주요 관점
- 고객지원 조직은 더 많은 상담원을 채용하는 구조에서, 불필요한 티켓 자체를 제거하는 AI 중심 운영 모델로 전환될 것이라고 본다.
- 기업용 AI는 기능(feature) 판매보다 end-to-end 워크플로우를 처리하는 agent 배치 형태로 진화하며, SaaS에서 GaaS(Agentic as a Service)로 이동한다고 본다.
- 지원·세일즈 업무 자동화에서는 비즈니스 맥락을 학습한 AI가 반복 문의를 즉시 처리하고, 인간 판단이 필요한 경우에만 에스컬레이션해야 한다고 본다.

## 주목할 포스트
- 지원팀의 반복 문의를 AI가 처리하고 사람은 예외 상황만 맡게 되는 운영 모델을 제시한 포스트 (출처: 포스트 16)
- SaaS에서 GaaS로의 전환과 Offlo.ai의 AI Support Agent 방향성을 설명한 포스트 (출처: 포스트 17)
