# Illia Polosukhin

**소속/직함**: Co-author of "Attention Is All You Need"; AI/agent system thinker
**업데이트**: 2026-04-17


## AX 관련성
기업 AX에서 agent를 실제 업무에 투입하려면 모델 성능보다 권한, 실행 경계, 보안 설계가 중요하다는 기준을 제공한다.

## 주요 관점
- AI를 'Final Operating System'으로 보며 search, chat, action, multi-agent coordination이 하나의 흐름으로 통합될 수 있다고 본다.
- 실제 agent 도입의 핵심은 reasoning quality만이 아니라 permission architecture라고 본다.
- Encrypted credentials, permission gate, WASM isolation, prompt injection prevention, virtualized execution space 같은 보안 구조가 agent 신뢰성의 핵심이라고 시사한다.
